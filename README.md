#🔤 上标字母转换工具

📋 简介

这是一个简洁的 Python 脚本，用于在普通文本和上标字母之间进行转换。特别适合 CTF 比赛、密码学题目、文本混淆等场景。

✨ 功能特性

🔄 双向转换：支持普通文本 ↔ 上标字母

🎯 简单易用：命令行参数，无需复杂配置

🚀 零依赖：纯 Python 标准库，开箱即用

📱 跨平台：支持 Windows、macOS、Linux
一、支持转化A-Z,a-z,0-9

二、参数含义示例 


 -s 转为上标 (Superscript)    

python super.py -s "TEXT"  

-b 转为普通 (Back to normal)  

python super.py -b "ᵀᴱˣᵀ"
